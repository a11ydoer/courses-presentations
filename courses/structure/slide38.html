Simple Skip to Main Content

  • Keyboard only users.
  • Screen readers users.

Features

  • Link (e.g. a) at the beginning of the page.
  • Visible when link receives keyboard focus:
    • Use CSS psuedo :focus and :active.
  • Place id on heading or main landmark identifying the start of the main content.
